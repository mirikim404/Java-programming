# Java 프로그래밍 및 실습 1

2026-1학기 Java 프로그래밍 및 실습 1 실습 코드 저장소입니다.

## 주차별 실습 내용

### Week 1 - 개발 환경 설정
- Eclipse IDE 설치 및 기본 사용법
- Java 프로젝트 생성 및 Hello World 출력

### Week 2 - 변수와 자료형 / 수식과 연산자
- **TypeConversion** : 자동/강제 형변환
- **CalcExample** : 기본 산술 연산
- **AssignmentIncDecOperator** : 대입 연산자, 증감 연산자
- **BitOperator** : 비트 연산자
- **LogicalOperator** : 논리 연산자
- **InfinityNaNExample** : Infinity, NaN 개념
- **FixedDepositCalculator** : 정기예금 이자 계산 실습
- **RobotReporter** : 변수 활용 출력 실습
- **OrderProgram** : 연산자 우선순위
- **Exercise** : 종합 연습 문제

### Week 3 - 조건문 / 반복문 / 메소드
- **MultiplicationTable** : 홀수/짝수 단 선택 구구단 출력
- **NestedLoop** : 중첩 반복문 연습
- **RecurringDeposit** : 정기적금 만기금액 및 세전/세후 이자 계산
- **Shape** : 메뉴 방식의 다양한 도형 그리기 (삼각형, 사각형, 크리스마스트리 등)
- **SuccessOrFail** : 조건문을 활용한 합격/불합격 판별
- **UpDownGame** : 랜덤 숫자 맞추기 업다운 게임
- **WhileSample** : while문 기본 연습
- **Wishlist** : n개의 위시리스트 입력 후 총 비용 출력

## 개발 환경
- Java 8
- Eclipse IDE
